Mutation Testing of Reinforcement Learning models at runtime.

ASASM

Some parameters which define the environment of a system may have not been taken into account while a model is being trained in that system using reinforced learning. This software aims to highlight the performance of a reinforced model under such changing factors.

Upload an environment with a model loaded. Then press generate to get a detailed report.
